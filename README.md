# Voice Recorder

A Sample web app that record your voice.
It uses React, Redux, TypeScript, IndexedDB, and [RecordRTC](http://recordrtc.org/).

[See the site here](https://voice-recorder-190010.appspot.com/)

## Local running
1. Clone the project.
2. Run `npm install`.
3. Run `npm run build:dev`.
4. Run `npm start`
